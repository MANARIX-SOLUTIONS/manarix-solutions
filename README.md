# Manarix Solutions - Landing Page

A modern, professional landing page for Manarix Solutions, a digital transformation consulting firm.

## Tech Stack

- **React 19** with Vite
- **Tailwind CSS v4** for styling
- Fully responsive design

## Getting Started

```bash
# Install dependencies (if not already done)
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Design

- **Colors**: Deep blue (#0B3D91), vibrant orange (#FF6F3C), light gray (#F5F5F5)
- **Typography**: Montserrat, Open Sans
- **Sections**: Hero, About, Services, Case Studies, Why Choose Us, CTA, Footer
